## API Reference

### Import

To use this component in your project, include the following import statement in your file.

```
import 'package:gluestack_ui/gluestack_ui.dart';
```

#### GSTab

GSTab can be used with help of 2 widgets, GSTabHeader & GSTabContent. Remember to ensure that parent widget is Stateful widget as we need to make use of GSTabController, to handle tab pages.
GSTabController is built on top of Flutter's [PageController](https://api.flutter.dev/flutter/widgets/PageController-class.html).
